# SavorAfrik-APP
SavorAfrik est une application web interactive qui permet de découvrir les plats traditionnels d'Afrique Centrale (Cameroun, Congo, Gabon, RCA, Tchad, Guinée Équatoriale) **Python** - Langage principal - **Streamlit** - Framework web - **Plotly** - Graphiques interactifs - **Requests** - API (TheMealDB)
